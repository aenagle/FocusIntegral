# FocusIntegral - Таймер фокусировки с интегралами

Приложение для фокусировки с проверкой внимательности через решение интегралов.

## Возможности

- ⏲️ Установка таймера работы (от 1 до 120 минут)
- 📝 Список задач (добавление/удаление)
- 🎯 Выбор текущей задачи (фиксируется на время работы)
- 🧠 Проверка внимательности: при нажатии "Отдохнуть" нужно решить интеграл
- 🔢 3 попытки на решение интеграла
- ☕ Автоматический отдых после завершения таймера (1/4 от времени работы)
- 📐 Различные типы интегралов:
  - Табличные интегралы
  - Определённые интегралы
  - Интегралы по шаблонам
  - Неберущиеся интегралы

## Сборка и запуск

### Требования

- CMake 3.16 или выше
- Qt 6 или Qt 5
- Компилятор с поддержкой C++17

### Сборка через CMake

```bash
# Клонируем репозиторий
git clone https://github.com/aenagle/FocusIntegral.git
cd FocusIntegral

# Создаём папку для сборки
mkdir build
cd build

# Генерируем файлы сборки
cmake ..

# Собираем проект
cmake --build . --config Release

# Запускаем
./FocusIntegral   # Linux/macOS
./Release/FocusIntegral.exe   # Windows
